How your data is collected
01
Discovery
Licensed dispensaries identified via California DCC license database and public maps APIs. No competitor is added without a verified DCC license number.
Nightly scan
02
Collection
Full menu data collected from publicly available cannabis market data sources. Pricing, SKU details, sale flags, discount labels, and lab values are captured per item.
2–5 AM Pacific
03
Normalization
Product names are normalized across dispensaries so "Friendly Farms OG 1g" and "FF OG Cartridge 1000mg" resolve to the same SKU for accurate cross-market comparison.
AI-assisted
04
Diff & Alert
Every new collection is compared to the prior snapshot. Price changes, new SKUs, removed products, and promo starts/ends generate alerts in real time.
Real-time events
05
Redundancy
A parallel fallback pipeline runs independently. If primary collection encounters an issue, fallback activates automatically — your data never goes stale.
Always-on backup
